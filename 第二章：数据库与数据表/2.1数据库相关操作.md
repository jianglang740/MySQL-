# 创建数据库：

* 在下载并配置好MySQL相关服务后，打开MySQL shell，可以通过输入以下操作来查看当前所有数据库：

```sql
shuw databases;
```
---
* 通过以下语法来创建一个数据库：

```sql
create database database_name;
```
---
* 通过输入以下语法来查看所创建的数据库的定义：
```sql
show create database database_name;
```
*通过输入以下语法来删除数据库：
```sql
drop database database_name;
```
